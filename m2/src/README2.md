# 思路

1. 找潜在分型
   方法一，可以通过找潜在点模式
   方法二，也可以通过特征分型合并的方式（只使用前包含后规则，不适用后包含前规则）
2. 找破坏
    如果出现了新高新低破坏了原来的分型
3. 找确认（方法一需要）
    查找第三特征来确认分型的有效性
4. 校验
    两种情况：1. 分型无缺口，确认前线段结束
    2. 分型有缺口，后面一个分型特征要用全包含处理（前包含后要处理，后包含前也要处理）
    2.1 后面一个特征序列无缺口，确认两个线段
    2.2 后面一个特征序列有缺口，确认一个线段


方法一和方法二要比较下，是不是结果是相同，不同的地方要认真对待
